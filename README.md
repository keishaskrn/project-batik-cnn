# Klasifikasi Motif Batik Indonesia Menggunakan CNN

## Deskripsi
Aplikasi klasifikasi motif batik Indonesia menggunakan Convolutional Neural Network (CNN) dan Flask.

## Fitur
- Upload gambar batik
- Prediksi motif batik
- Model CNN TensorFlow
- Web Interface Flask

## Dataset
Indonesian Batik Motifs Dataset (Kaggle)

## Teknologi
- Python
- TensorFlow
- Keras
- Flask
- NumPy
- Pillow

## Cara Menjalankan

Install dependency:

pip install -r requirements.txt

Training model:

python train_model.py

Menjalankan web:

python app.py
